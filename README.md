# Модель для предсказания биологической активности соединений против вирусов

#### Структура репозитория^

train.csv, test.csv, sample_submission.csv — исходные данные

solution.ipynb — код решения

submission.csv — финальные предсказания

README.md — этот файл

presentation.pdf — презентация проекта

## Целевые переменные

IC50 (мкМ) — концентрация, подавляющая 50% вируса

CC50 (мкМ) — концентрация, токсичная для 50% клеток

SI = CC50 / IC50 — индекс селективности

## Основные сложности

Сильно скошенные распределения с тяжелыми правыми хвостами

Выбросы: IC50 — 107 из 751 по IQR

Пропуски в 12 колонках (зарядовые дескрипторы, BCUT2D) — XGBoost обрабатывает NaN самостоятельно

15 константных fr-признаков (всегда 0) удалены

## Трансформации

log1p для IC50 и CC50 (симметризация, стабилизация дисперсии)

SI не обучается напрямую, вычисляется как CC50 / IC50

Защита от деления на ноль: IC50 < 0.001 → 0.001

## Отбор признаков

IC50: 50 лучших признаков

CC50: 30 лучших признаков

## Работа с выбросами

Удалены топ-8 худших остатков с каждого таргета (15 выбросов, 1 пересечение)

## Проблема: модель систематически завышала малые значения CC50 (предсказывала 265 вместо 1)

## Взвешенное обучение

Распределение весов по квартилям:

0-25%: вес 0.61

25-50%: вес 0.23

50-75%: вес 0.21

75-100%: вес 0.68

Края распределения получают в 3-5 раз больший вес

Гиперпараметры финальной модели

- learning_rate: 0.01

- n_estimators: 700

- max_depth: 4

- subsample: 0.6-0.7

- colsample_bytree: 0.7

- reg_alpha (CC50): 0.3

### Применение

Модель подходит для первичного скрининга: из 1000 соединений химики могут сосредоточиться на 200-300 наиболее перспективных, отсеивая заведомо неактивные или токсичные варианты.

Ограничения
Модель обучена на 751 молекуле одного химического класса. Для соединений с принципиально новой структурой точность может быть ниже. Рекомендуется экспериментальная валидация на независимом наборе перед внедрением.
